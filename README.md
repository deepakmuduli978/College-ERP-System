# College-ERP-System
The College ERP System is a menu-driven application developed using C programming language. It helps manage:  👨‍🎓 Student Records  👩‍🏫 Staff Records  📅 Timetable Management  This system demonstrates the use of structures, arrays, functions, and string handling in C.
ERP (Enterprise Resource Planning) systems help organizations manage data in an organized manner. In colleges, maintaining student records manually can be time-consuming and prone to errors.

This College ERP System automates student data management using C programming concepts such as:

Structures

Functions

File Handling

Loops & Conditional Statements
